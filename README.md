# Hands-On da disciplina de Algoritmos Quânticos

Esse repositório foi criado exclusivamente para desenvolver as atividades do curso de especialização em Computação Quântica - SENAI CIMATEC.

- [Grover](./Grover)
- [Problema do Caixeiro Viajante com QAOA](./QAOA/)

## Recomendações Gerais

- Para ambas as atividades desenvolvidas, o material deve estar bem organizado, desenvolver explicações claras com um referencial teórico bem elaborado, detalhando o funcionamento do algoritmo e os conceitos fundamentais da mecânica quântica utilizados;
- O código deve deve estar bem organizado e comentado, explicando qual o objetivo de cada etapa da implementação
- Especificamente para o *QAOA*, o problema do Caixeiro Viajante pode ser implementado para *4 cidades no Colab*
